# Solutions in C

This repo is basically where I’m practicing C.

I’m adding programs as I learn different topics — starting from basics and slowly moving to more important concepts. It’s not perfectly structured or anything, just consistent practice and improvement.

---

## What’s inside

You’ll find programs related to:

* basic syntax and input/output
* if-else and switch
* loops
* arrays
* functions
* pointers

Most of the code is simple and written in a way that I can understand later when I revisit it.

---

## Why this repo?

Honestly, just to get better at coding.

I come from a non-heavy coding background, so this is me trying to build a strong base step by step. Instead of just watching or reading, I’m actually solving and saving everything here.

---

## How to run

If you want to try any program:

```
gcc filename.c -o output
./output
```

That’s it.
